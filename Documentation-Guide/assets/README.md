# Assets Folder

This folder contains static assets used in documentation templates.

## Required Files

### nestle-logo.png
- **Purpose**: Client logo displayed on cover pages
- **Size**: Recommended 200x100 pixels or similar aspect ratio
- **Location**: Top-right corner of cover page

## Usage

Templates reference assets using relative paths:
```html
<img src="assets/nestle-logo.png" alt="Nestle" class="logo">
```

## Notes for Other Projects

When using these templates for a different client:
1. Replace `nestle-logo.png` with the client's logo
2. Update the `alt` text in HTML templates
3. Adjust CSS `.logo` class dimensions if needed
